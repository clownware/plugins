Construction notes

Source of truth: official Figma vector outline (assets/logo/emblem-white.svg / emblem-black.svg).
The stroke construction is ONE continuous path — the mark never lifts the pen:
  bar cap, crossfade diagonal, bottom bar, counter-diagonal, stem, stem cap.
  #mark path length 82.12 user units (stroke-dashoffset timing base).
Stroke width 15.578 px in the 145 x 162 frame, normalized to 2 user units.
X crossing at (85.11, 71.79) px. Corner radii (centerline): 15.9 / 12.4 / 12.75 / 12.8.
Fidelity vs official vector: IoU 0.991 at 580 px render; residuals are corner-curve
differences (Figma two-cubic corners vs circular arcs), sub-pixel at display sizes.
Filled outlines cannot animate via stroke-dashoffset; the stroke construction exists
so the motion idiom (draw-on, sting) runs on real path geometry.
