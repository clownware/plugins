Square icon-only button for toolbars and nav — ghost by default, supports an `active` toggled state.

```jsx
<IconButton label="Play" variant="ghost"><PlayIcon/></IconButton>
<IconButton label="Filter" active><FilterIcon/></IconButton>
```

Variants: `ghost`, `outline`, `solid`. Sizes match Button (32/40/48). Always pass `label` for a11y.
